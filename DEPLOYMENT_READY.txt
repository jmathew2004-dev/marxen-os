🛠️ HELPFUL COMMANDS:

# Using Makefile (most convenient)
make help          # Show all commands
make dev           # Start development servers
make build         # Build frontend
make docker-up     # Start all Docker services
make test          # Run tests
make migrate       # Run database migrations

# Using deploy script
./deploy.sh        # Interactive menu
./deploy.sh docker # Deploy with Docker
./deploy.sh build  # Build only (verify)

# Manual commands
cd backend && npm run dev    # Start backend server
cd frontend && npm run dev   # Start frontend server
npm run migrate              # Run migrations
npm run build               # Build frontend

📊 PRE-DEPLOYMENT CHECKLIST:

Before deploying to production:
□ All code committed to git
□ Frontend builds successfully (npm run build)
□ Backend syntax valid (npm run lint)
□ .env.production configured with real values
□ JWT_SECRET is strong (32+ characters)
□ CLAUDE_API_KEY set (or feature disabled)
□ FRONTEND_URL matches your domain
□ Database backups planned
□ SSL/HTTPS setup planned
