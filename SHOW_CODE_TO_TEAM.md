## 💡 Tips for Presenting

### Tip 1: Use Presentation Mode
```
Press: Ctrl+K Ctrl+P (Mac: Cmd+K Cmd+P)
Effect: Code becomes HUGE and readable from across room!
```

### Tip 2: Split Screen
```
Press: Ctrl+\ (Mac: Cmd+\)
Effect: Two files side-by-side - show frontend AND backend!
```

### Tip 3: Search Code
```
Press: Ctrl+F (Mac: Cmd+F)
Find: Function names, variables, etc
Perfect: For showing where things are used!
```

### Tip 4: Go to Definition
```
Click: Ctrl+Click any function/variable
Effect: Jump to where it's defined
Great: For following code flow!
```

### Tip 5: Search All Files
```
Press: Ctrl+Shift+F (Mac: Cmd+Shift+F)
Effect: Search entire codebase
Use: To find implementations across files!
```
